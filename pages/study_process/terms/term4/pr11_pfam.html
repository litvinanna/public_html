Сначала была получена информацию об архитектуре всехпоследовательностей, входящих в данное семейство – таблица соответствия последовательностей и доменов (лист PF00917). 

python swisspfam-to-xls.py -i /srv/databases/pfam/swisspfam.gz -z -p PF00917 -o PF00917.xls

На основе данного списка была составление сводная талица: строки - AC, столбцы - PFAM ID (лист svod). В ней можно было отобрать нужные архитектуры. Были выбраны следующие две.

Архитектура 2.

По сводной таблице с помощью формул были выбраны белки, содержащие нужную комбинацию доменов. Далее список выбранных белков был перенесен на новый лист вместе с номером архитектуры (1 или 2) (лист choose).

Также по списку AC белков из семейства PF00917 была получена информация о таксономии. Сначала были скачаны uniprot записи для всех белков. Затем из записей извлечена информация о таксономии с помощью команды: 

python uniprot_to_taxonomy.py -i PF00917-uniprot -o PF00917-tax.xls

Таксономия была также добавлена в общий файл на новую страницу (лист tax). К последовательностям выбранных архитектур была добавлена таксономия с помощью функции (VLOOKUP)

Далее на основании таксономии и архитектуры были выбраны последовательности для выравнивания. Домен MATH в двух выбранные архитектрах оказался широко представлен среди растений и двух подтипов многоклеочных животных (Metazoa) - некоторых Ecdyzozoa (Nematoda, Arthroda) и позвоночныx (Vertebrata, особенно черепных Craniata).

Выбранные последовательности были снабжены дополнительными идентификаторами: номером архитектуры (1 или 2) и буквами таксона. Были введены обозначения:

  • Em - подтип Embriophyta, тип Viridiplantae - хорошо представленный подтип растений
  • Cr - подкласс Craniata, класс Vertebrata - черепные животные
  • Ne - Nematoda - нематоды
  • Ar - Arthropoda - членистоногие, нематод и членистоногих можно рассматривать как основных представителей подтипа Ecdyzozoa.

С помощью скрипта из общего выравнивания домена MATH из всех белков, его содержащих, были вытащены выбранные фрагменты. В полученном выравнивании были также исправлены названия последовательностей в соответствии с идентификаторами. Варавнивание было очищено - удалены плохие концевые участки, также последовательности, которые были совсем плохо выровнены.

По выравниванию было постороено филогенетическое дерево в программе MEGA методом maximum likehood с методов проверки bootstrap со 100 репликами. Дерево анализировалось в сервисе ITOL.

Раскрашенное дерево, бутстрап поддержка ветвей обозначена через их толщину.

Рамочками выделены спорные участки дерева. Ветви раскрашены, если они образованы одно архитектурой одного таксона, кроме синей ветви, где наблюдается стойкое "смешение" первой архитектуры среди Metazoa.

В целом можно сказать, что архитектуры возникли раньше, чем разделились представленные таксоны. Хотя в растения являются очень далекой группой по отношению к животным. По крайнем мере во второй архитектуре наблюдается ровное разбиение по таксонам. Это значит, что домен развивался в каждом таксоне независимо.

Понять куда данное дерево может быть укоренено сложно. Ближе к показанному здесь корню находятся ветви с нулевой поддержкой. Для адекватной оценки того, когда разошлись данные архитектуры нужно проанализировать больше данных. По данной выборке можно только сказать, что домены в разных архитектурах развивались более менее независимо.
