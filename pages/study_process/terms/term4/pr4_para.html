Дерево по 16s rRNA

Филогенетическое дерево тех же бактерий было постороено с использованием нуклеотидных последовательностей гена 16s rRNA (субъединица рибосомальной РНК).

Геномы многих бактерий лежат в базе NCBI. Каждый геном состоит нескольких файлов, содержащих различную информацию о генах, его свойствах, последовательностях. В файле .frn лежат последовательности всех РНК клетки. Для каждой бактерии в базе была найдена наиболее близкая (некоторый штамм этого вида), и из указанного файла была скопирована последовательность 16s rRNA. Ген 16s rRNA часто представлен в геноме несколько раз (Wikipedia), для анализы была выбрана первая последовательность из списка. Файл с последовательностями [fasta] - таб.1.

Для Streptococcus pyogenes serotype M1 в банке не оказалось последовательности 16s rRNA, поэтому был взят ген малой субъединицы РНК, который содержит 16S. После выравнивания было обнаружено, что эта малая субъединица выровнялась с другими 16S без лишних хвостов, возможно она содержала только последовательность 16S.

Последовательности были выровнены в Jalview методом Muscle with Defaults. Выранивание [fasta].

Филогенетичсекое дерево было построено в программе MEGA5 методом maximum likehood - рис.1..

Дерево, реконструированное по 16s rRNA полностью совпадает с принятой филогенией. По белкам верное дерево построить не удалось.

Последовательность 16s rRNA есть во всех бактериях и очень консервативна, так как она работает в таком "базовом" клеточном процессе как трансляция. Известно, что рибосомы и, соотвественно рРНК, устроены похоже у всех организмов, а в пределах бактерий они очень похожи. Поэтому происходящие мутации в данном гене: случаются с одинаковой скоростью во всех организах и ветках дерева, и случаются редко, поэтому соответствуют филогении. Такие консервативные последовательности обычно и используют для реконструкции филогении достаточно далеких организмов.

Ортологи и паралоги

В протеомах бактерий были найдены гомологи белка CLPX_BACSU. Для этого я скачала протемы своих бактерий в формате fasta, записала их в один файл с помощью cat proteom >> proteims, полученный файл [fasta]. Затем в помощью blastp провела поиск белка по базе протеомов с порогом e-value 10^(-10). 

blastp -query clpx.fasta -subject all_proteomes.fasta -outfmt '6 sseqid sseq' -evalue 1e-10 -out found-1.fasta

Полученные последовательности были выровнены в Jalview. Затем построено филогенетическое дерево методом наибольшего правдоподобия.

На рис.3. отмечены паралоги и ортологи. Паралоги — гомологичные белки из одного организма. Ортологи — гомологичные белки из разных организмов, разделение их общего предка на линии, ведущие к ним, произошло в результате видообразования.

Зелёным выделены две группы ортологов, синий и жёлтым — две группы паралогов.

Соответственно, в организмах FINM2 и STRP1 произошло по одной дупликации. Каждая группа ортологов стала таковыми в результате видообразования, у общего предка был 'предок' данных белков, который затем при разделени видов начал эволюционировать в каждом организме отдельно. В группах ортологов много белков с одинаковыми названиями, что говорит о том, что это действительно схожие белки, выполняющие сейчас похожие функции в разных бактериях.

Чтобы обнаружить больше ортологов и паралогов blastp был запущен с порогом evalue 10^(-5), белки были выравнены muscle полученный файл [fasta]. Из последовательностей построено дерево методов наибольшего правдоподобия.

На рис.4. разноцветными рамочками показаны некоторые группы паралогов. Они возникли, скорее всего, в результате генных дупликаций в каждом организме.
