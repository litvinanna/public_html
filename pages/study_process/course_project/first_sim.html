pdb2gmx -f 6pti.pdb -water tip3p
You will be prompted for the force field (select “6” for Amber99SB-ILDN. Собственно, название поля: 
 6: AMBER99SB protein, nucleic AMBER94 (Hornak et al., Proteins 65, 712-725, 2006)
command will produce three files: conf.gro contains coordinates with hydrogens, topol.top is the topology, and posre.itp contains a list of position restraints 
editconf -f conf.gro -d 0.75 -o box.gro

genbox -cp box.gro -cs spc216.gro -p topol.top -o solvated.gro
touch ions.mdp
 grompp -f ions.mdp -p topol.top -c solvated.gro -o ions.tpr
 genion -s ions.tpr -neutral -conc 0.1 -p topol.top -o ions.gro
